About


This class is Web Application Programming and Hacking taught by Dr. Phu Phung at the University of Dayton. It teaches the process of building web applications with an emphasis on "thinking like a hacker" in order to create secure applications during the development cycle. Often times security vulnerabilities are patched after the fact which can lead to data leaks, loss of money and a loss of trust in your consumers. In order to prevent taught the developers should put security at the forefront instead of an afterthought.
